## Hal Vipan - 22200387

This is a CMake project, to run the project ensure CMake is installed using:
```
which cmake
```
To generate build files run:
```
cmake -B build
```
---
To build the server project:
```
cmake --build build --target CO650_TASK2_SERVER
```
To execute the server project:
```
./build/CO650_TASK2_SERVER
```
---
To build the client project:
```
cmake --build build --target CO650_TASK2_CLIENT
```
To execute the client project:
```
./build/CO650_TASK2_CLIENT
```
